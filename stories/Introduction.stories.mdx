# Root

Open source components for web-based audio applications

- A component library for making audio apps on the web
- Makes the Web Audio API easier to use
- Provides accessible, reusable UI components


## Installation
To install the latest stable version.

`npm install @potato/root`

## Make some sound with your MIDI Keyboard

  <root-midi></root-midi>
  <root-connect>
